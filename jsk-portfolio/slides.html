DivyaDesam Explorer

Mobile App

A cross-platform mobile reference app cataloguing all 108 Divya Desam temples — sacred Vishnu temples across South India. Features complete temple data, poet-saint biographies, and Nalayira Divya Prabandham verses with Tamil text, transliteration, and English meanings.

React Native Expo TypeScript Expo Router
Key Features
  • 108 temples across 6 geographic regions
  • 12 Azhwar poet-saints with biographical data
  • 158 temple-verse combinations with Tamil/English
  • Offline-first — all data embedded locally
  • Pre-built APK available

LC Compliance Checker

FinTech / Trade

A full-stack application for validating Letter of Credit documents against UCP600 international trade rules. Extracts data from LCs, invoices, bills of lading, and certificates using PDF parsing and OCR.

FastAPI Python React PyMuPDF Tesseract OCR RapidFuzz
Key Features
  • PDF extraction with OCR fallback
  • Fuzzy matching for field comparison
  • UCP600 compliance rule engine
  • Severity-graded compliance reports
  • Multi-document session management
